Restyling de marca

Diseño gráfico

Cliente: Taller DosGrillos

Me fue solicitada la creación de tarjetas de presentación para el taller de adornos artesanales Dos Grillos.
Presenté, además, un esquema de hojas para presupuesto y facturación que gustaron mucho al cliente y también fueron aceptadas.
El primer paso fue hacer un restyling del logotipo del comercio. Líneas discretas y minimalistas resultaron adecuadas para reflejar la seriedad de la marca.
Este logotipo iría impreso en el packaging de cada producto elaborado. Los materiales que solían utilizar para envolver sus productos eran cajas de cartón en crudo y papel madera como envoltorio de elementos pequeños. Presenté un esquema donde el cliente pudo apreciar el aspecto del logo impreso en los materiales de empaquetado.
Tanto para las tarjetas como para las hojas de administración, usé un esquema de colores sobrio pero luminoso. La información que me fue requerida presentar quedó muy bien distribuida de manera clara y con una visual atractiva y profesional.

Construcción de sitio web

Desarrollo web

para: Taller DosGrillos

Taller DosGrillos se dedica a elaborar muebles artesanales de metal y madera y productos de decoración.
Inicialmente, su modus de venta se sostenía mas que nada de forma presencial en su showroom. Pero las necesidades de crecimiento y actualización del negocio llevaron a sus dueños a plantearse la idea de establecerse en la web de forma adecuada y profesional.
El sitio creado posee una estética simple y elegante. Con secciones bien diferenciadas para facilitar la navegación de sus visitantes.

El visitante podrá apreciar cada uno de sus productos de forma detallada, ademas de visualizar una descripción de cada artículo. Se podrá comprar el producto de interés a través de un link a la plataforma de ventas de Mercado Libre. También se puede optar por contactar directamente con la oficina de ventas del taller a través de un mail para opciones de personalización de un artículo.

Calculador de ganancias con Dolar-Loop

Software - Aplicación Web

Angular, CSS, HTML, Bootstrap, Github Pages, API Rest

La aplicación realiza el cálculo de márgen ganado después de una operacion de compra y venta de la divisa estadounidense aprovechando la diferencia en las cotizaciones de esa moneda en Argentina.

Funcionamiento
El usuario completa los campos con el capital en pesos argentinos (ARS) que simula invertir. Se reflejarán mediante etiquetas y de forma simultanea al ingreso de datos, los resultados de tal inversión.

Sobre el proyecto
La aplicación fue contruída sobre el framework de Angular.
Para la obtención de información actualizada de las cotizaciones se consumen datos de la API pública de criptoya.com
El resultado responsive se logró usando la librería Bootstrap.

Tecnologías utilizadas

Github Pages (Alojamiento y Deploy)
Framework Angular
CSS, HTML y Typescript
Librería Bootstrap
API de criptoya.com
